Twierdzenie Pitagorasa

a² + b² = c²

Suma kwadratów długości przyprostokątnych trójkąta prostokątnego jest równa kwadratowi długości przeciwprostokątnej tego kwadratu. a,b,c ≠ 0

Wzór równania kwadratowego

Wzór, gdzie ax² + bx + c = 0 i a ≠ 0. Wielkości a, b, c są znane jako współczynniki, kolejno: kwadratowy, liniowy i stały bądź wyraz wolny. Założenie a ≠ 0 oznacza, że do równań kwadratowych nie zalicza się równań liniowych.
